# Resume Generation Portal

A web-based Resume Builder System that allows users to create, manage, and generate resumes dynamically using pre-defined templates.

## 🚀 Features
- User Registration & Login Authentication
- Candidate Profile Management
- Resume Generation (Single & Double Column)
- Dynamic Data Fetching from Database
- Print / Export Resume as PDF

## 🛠️ Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

## 📂 Project Structure
- index.php – Landing Page
- register.php – User Registration
- login.php – Login System
- dashboard.php – Navigation Hub
- candidate_profile.php – Profile Management
- resume_single.php – Single Column Resume
- resume_double.php – Double Column Resume
- db.php – Database Connection
- resume.sql – Database Backup

## 🖥️ How to Run
1. Install XAMPP/WAMP
2. Place project in `htdocs`
3. Import `resume.sql` into phpMyAdmin
4. Run `localhost/project-folder`
